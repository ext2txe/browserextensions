# Quick Start Guide

## Install the Extension in 3 Steps

### 1. Load the Extension
- Open Firefox
- Type `about:debugging#/runtime/this-firefox` in the address bar
- Click "Load Temporary Add-on"
- Navigate to this folder and select `manifest.json`

### 2. Configure Your Keywords
- Click the extension icon in your Firefox toolbar (green icon)
- Add your keywords:
  - Type "India" and click "Add"
  - Type "Saudi Arabia" and click "Add"
- (Optional) Change the highlight color using the color picker
- Click "Save Settings"

### 3. Visit Upwork
- Go to https://www.upwork.com/nx/find-work/
- Jobs containing "India" or "Saudi Arabia" will now be highlighted!

## Tips

- **Case-insensitive**: "india", "India", and "INDIA" all match
- **Partial matching**: The keyword "India" will match "Indian" or "India-based"
- **Edit keywords**: Click on any keyword in the list to edit it
- **Multiple keywords**: Add as many as you want - jobs matching ANY keyword will be highlighted
- **Live updates**: The extension works with Upwork's infinite scroll

## Default Settings

- **Default color**: Light gray (#d3d3d3)
- **Default keywords**: None (you need to add your own)

Enjoy your highlighted job search!
