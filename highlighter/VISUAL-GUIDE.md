# Where to Find the Settings Button

After installing the extension and visiting the Upwork job search page, you'll see:

## Settings Button Location

The settings button appears in the **bottom-right corner** of the Upwork page:

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│  Upwork Job Listings Page                          │
│                                                     │
│  ┌────────────────────────────────────┐            │
│  │ Job 1 (highlighted if matches)     │            │
│  └────────────────────────────────────┘            │
│                                                     │
│  ┌────────────────────────────────────┐            │
│  │ Job 2                              │            │
│  └────────────────────────────────────┘            │
│                                                     │
│  ┌────────────────────────────────────┐            │
│  │ Job 3 (highlighted if matches)     │            │
│  └────────────────────────────────────┘            │
│                                                     │
│                                                     │
│                          ┌──────────────────────┐  │
│                          │ ⚙️ Highlighter      │  │ ← Click here!
│                          │    Settings         │  │
│                          └──────────────────────┘  │
└─────────────────────────────────────────────────────┘
```

## What the Button Looks Like

- **Color**: Green background with white text
- **Icon**: ⚙️ (gear emoji)
- **Text**: "Highlighter Settings"
- **Location**: Fixed position at bottom-right, always visible even when scrolling

## When You Click It

A settings panel will slide up from the button showing:
- Color picker for highlight color
- Text field to add new keywords
- List of existing keywords (editable)
- Save button to apply changes
- Close button to hide the panel

## Quick Demo

1. Visit: https://www.upwork.com/nx/find-work/
2. Look at the bottom-right corner
3. Click the green "⚙️ Highlighter Settings" button
4. Add keywords and save!

The button stays on the page while you browse jobs, so you can adjust settings anytime.
